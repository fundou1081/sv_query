## 开发纪律

1. **金标准测试优先** - 先写测试，再写代码
2. **保持 AST 唯一性** - 禁止正则分析源码
3. **标注置信度** - 每个 API 返回带 confidence
4. **文档同步** - 代码变更同步更新文档
5. **测试驱动** - 用测试用例验证功能
